How to League 101

There are many aspects to learning this game but this is

a basic written tutorial on how to play League of Legends.

Choosing a map

There are 3 main maps you can play on and some special game modes appearances that Riot will occasionally put on.

There are Summoners Rift (5v5), Twisted Treeline (3v3),and ARAM(All Random All Mid) (5v5)

You can either play with friends or get teamed up with strangers
